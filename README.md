# Clean architecture plus MVVM with React

## Introduction

This is a sample project to show how to implement the clean architecture with MVVM in React.

## How to run

```bash
$ pnpm install
$ pnpm dev
```

## How to test

```bash
$ pnpm test
```

## How to build

```bash
$ pnpm build
```

## How to lint

```bash
$ pnpm lint
```

## How to format

```bash
$ pnpm format
```
